# 🚀 VidyaJobs.AI 
**Your One-Stop Portal for Govt Jobs & Exam Results**

VidyaJobs.AI is a professional web-based platform designed to help students track the latest career opportunities in Bihar and across India.

### ✨ Key Features:
* 🏛️ **Latest Govt Jobs:** Real-time updates on state and central vacancies.
* 🚂 **Railway Special:** Dedicated section for RRB/RRC updates.
* 📄 **Check Results:** Direct links to official result portals (BSEB, CSBC, etc.).
* 📚 **AI Study Planner:** Personalized prep schedules for exams like SSC GD.

### 🛠️ Built With:
* HTML5 & CSS3 (Modern Responsive UI)
* JavaScript (Dynamic Content Fetching)
* Google Search Integration
